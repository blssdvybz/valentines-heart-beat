A Valentine's Day themed interactive web application.

## Features
- **Interactive Gameplay**: Engage in a fun heart-catching mini-game.
- **Dodging Button**: A playful "No" button that dodges your cursor.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Construction Theme**: Unique "Build Hearts Together" aesthetic.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Radix UI
- **Backend**: Express
- **Database**: Drizzle ORM (PostgreSQL)
- **Animation**: Framer Motion

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
